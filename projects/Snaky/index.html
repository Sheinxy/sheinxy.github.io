Snaky

Snaky is a small Discord bot that I code on my spare time. This bot is aimed at allowing users to create themselves custom commands, especially emotes, as well as some other few things.
Snaky was created in november 2017 as a bot for my friends' discord server. At first, it didn't have much success, but with some time and new commands, it started getting some use.

-help (name) :: Hellow! I'm Snaky, your friendly neighborhood Snake Robot ! Any question to ask me ?

-clear <number> :: I can help clearing your junk ! Just tell me how many messages I have to delete and I will do it! UwU

-say (message) :: I will repeat anything that you say! :3 Like a small parrot! (But I'm a Snake, so please, don't say that I'm a parrot ;W;)

-commands :: I can display so many commands ! You must be lost with all of them, let me refresh your memory to help you! :3

-define <name> <command> :: Wanna add a command to the server ? ^w^\nMeta values $Name(Arguments) :: Author, Gif, Arguments, Mentions, Api, Random

-undefine <name> :: Are you unsatisfied by an command ? I can delete it for you ! :3

-quote <scope> :: Let me give you a quote that one of you added here ! Use -quote server for a server quote, -quote public for a public one ! ^w^

-add_quote <scope> <quote> :: Wanna add a quote to my quote list? :3 Use -addQuote server to add a server quote, -quote public to add a public one ! ^w^

-prefix :: You don't remember what prefixes are available here ? Don't worry, I will reming you ! ^w^

-add_prefix <prefix> :: Do you want to add a new prefix for this server ? ^w^

-del_prefix <prefix> :: Do you want to remove a prefix from this server ? ^w^

-goodbye <channel_ID | default | none> :: Where do you want me to say when someone joins or leaves ? uwu

-disable <role_ID> :: Make sure that people with a certain role won't be able to use a certain command òwó

-enable <role_ID> :: Make sure that people with a certain role will be able to use a certain command òwó
